# iCodex 0.2.02 更新说明

- 输入栏中的权限选择器会根据ui收缩；点击仍可打开菜单，图标颜色会对应当前权限。
- 权限配置现在直接使用 Codex CLI 的 `approvalPolicy` 与 `sandbox` 返回值：支持只读、请求批准、替我审批和完全访问。打开已有对话时会恢复该对话保存的权限；新会话和后续回合会沿用当前选择并向 CLI 发送相同配置。

## Codex CLI

- 本地和云端启动器优先查找启动目录及子目录中的 `codex.exe`。
- 首次找不到本地 CLI 时，可输入完整路径；直接回车会寻找系统 CLI。
- 选定路径保存在 `.ip-public/codex-cli-path.txt`

## 模型与对话配置

- 可记住手动选择的模型，作为同一提供商的新对话默认模型。
- 打开已有 Codex 对话时，恢复该对话的模型和推理强度，避免被全局默认值覆盖。
- 推理强度从 CLI 的模型能力目录读取，不再使用固定前端列表。
- 5.6 模型提供轻度、中、高、极高和极高快速预设；快速预设会同时使用 `xhigh` 与 Fast 服务层。

## 界面与本地化

- 移除模型搜索框。
- 补齐推理强度中文：无、最低、轻度、中、高、极高，以及“更快消耗使用额度”。
